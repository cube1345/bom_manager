# BOM Manager 项目介绍

`BOM Manager` 是一个面向电子硬件团队的元器件与 PCB 数据管理工具，帮助你把分散的物料信息集中管理，并快速完成统计、导入导出和项目协作分享。

## 主要作用

1. 统一管理元器件信息  
支持新增、编辑、删除元器件，记录型号、类别、平台、单价、库存/用量等关键字段，减少 Excel 多版本混乱。

2. 统一管理元器件类型  
可维护元器件类型清单，新增类型后可直接在元器件表单中使用，数据自动联动更新。

3. 管理项目与 PCB 结构  
一个项目下可维护多个 PCB；每个 PCB 可独立维护所需元器件清单，实现分层管理。

4. 自动统计需求关系  
可从项目/PCB 维度统计需要哪些元器件，也可从元器件维度反查被哪些 PCB 使用，便于采购和备料。

5. 批量导入与智能映射  
支持 JSON/CSV/Excel 批量导入，且可自由映射 Excel 列名到系统字段，适配不同团队模板。

6. 一键导出 JSON / Excel  
支持将当前数据导出为标准 JSON 或 Excel，便于备份、归档、跨系统共享。

7. 持续化存储与可配置路径  
数据可自动持久化保存，并支持自定义存储目录，便于团队按规范管理数据。


## 适用场景

- 研发团队进行 BOM 建立与维护  
- 采购团队进行物料统计与成本核算  
- 项目经理进行多项目、多 PCB 的物料统筹  
- 企业内部进行工具化部署与版本发布
